# Avito Auto AI Answer

A Chrome extension that automatically replies to customer messages on Avito using AI.

![License](https://img.shields.io/badge/license-MIT-blue)

---

## 🚀 Features
- Automatic replies to incoming messages  
- Saves time when communicating with customers  
- Easy to install and use  
- Works directly in Chrome  

---

## 📦 Installation

### 1. Clone the repository
```bash
cd C:/
git clone https://github.com/chelobekPython/Avito-auto-ai-answer
```
### 2. Load the extension in Chrome
Open: chrome://extensions/
Enable Developer mode
Click Load unpacked
Select the project folder

### ⚙️ Usage

After installation, the extension will start working automatically on Avito and reply to messages.
